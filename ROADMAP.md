# Roadmap

## MVP

- Portfolio React/Vite
- Thème Onyx / Candy Blue
- Pages Accueil, Projets, Détail projet, À propos et Contact
- Avatar réel
- Données centralisées
- Documentation de base
- Build GitHub Pages

## V1

- Ajouter le CV officiel
- Ajouter des captures réelles validées pour Gold Sniper et ClipWave
- Ajouter plus de projets uniquement après revue de code
- Ajouter des métriques Lighthouse

## V2

- Ajouter un blog technique
- Ajouter une version anglaise
- Ajouter des études de cas plus longues
- Ajouter un domaine personnalisé

## Futur panel admin

Un panel admin est hors MVP. Il pourra être étudié plus tard avec authentification, rôles, audit et stockage externe.

## À garder

- Ne pas exposer d'adresse complète.
- Ne pas exposer de secrets.
- Ne pas inventer de fonctionnalités projet.
- Garder les descriptions liées au code réel.
